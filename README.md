## 📖 About

**XIAO Coin** is a community-driven cryptocurrency inspired by the charm and resilience of the giant panda. The site serves as a centralized digital presence for the $XIAO community — showcasing tokenomics, how-to-buy guides, project stats, and a chill vibe for all holders.

### 🎯 Mission

In the chaotic world of meme coins, $XIAO aims to bring balance and peace (just like a panda in a bamboo forest). We focus on building a sustainable ecosystem with fair distribution and community-led initiatives.

- **For Investors:** Transparent tokenomics and easy-to-follow buying guides.
- **For the Community:** Interactive features, gallery, and music player to enjoy the ride.
- **For the Developers:** A clean, optimized codebase ready for high-traffic launches.

## ✨ Features

### 🏠 Homepage Sections

| Section | Description |
|---|---|
| **Hero Section** | Full-screen video background with $XIAO branding, tagline, and quick-action buttons to Buy or check Charts. |
| **About Section** | Introduction to the "Super Duper Panda" philosophy with a custom video machine animation and responsive description. |
| **How to Buy** | Step-by-step guide for setting up a Solana wallet, adding SOL, and swapping for $XIAO on DEX platforms. |
| **Tokenomics** | Dynamic Donut Chart (Chart.js) visualizing the 1 Billion supply distribution across Ecosystem, Airdrops, and more. |
| **Stats Counter** | Animated intersection-observed counters for Total Supply, Tax, and Buy Back percentages. |
| **Gallery** | Visual showcase of $XIAO-themed artwork with hover-scale effects. |
| **Partnership** | Grid of supporting platforms including CoinMarketCap, CoinGecko, and top exchanges. |
| **Music Player** | Persistent, custom-styled music player with a curated playlist, progress tracking, and autoplay capabilities. |

### 📂 Technical Highlights

- **Custom Music Player** — Built with native JS and CSS, featuring a rotating vinyl effect and playlist management.
- **Dynamic Charts** — Uses Chart.js for beautiful, responsive data visualization.
- **Intersection Observer** — Triggers animations (like counters) only when they enter the user's viewport.
- **Responsive Design** — Fully adaptive layout using Bootstrap 5 and custom CSS media queries.
- **Infinite Carousel** — Smooth, automated image sliders for gallery and branding.
- **Loading Screen** — Animated entry sequence to ensure all assets are ready before display.

2. **Open the project**
